Css is nothing but Cascading Style Sheet.
                     It is used Style or design the html document.
                  

                     The syntax for the css for styling is --
                     selector{
                        proprty:value;
                     }
                  

                    There are 3 ways to attaching css styling to html.
                    1.Inline-Which gives styling to inline element using  tag.
                    2.Internal-Which gives the styling in the below head tag and above body tag using  tag.
                    3.External-which gives the styling in the external css file which is having .css extension.
                    The link tag is used connect from html file to css file using an attribute href.
                  

                    There are 5 types of selectors we have They are:
                    1.Simple selector
                    2.Combinator selector
                    3.Pseudo-classes
                    4.Pseudo-elements
                    5.Attribute selector
                  

                    The difference between id and class is -
                    class-class is used for multiple elements and we can give mutiple class names to one particular one.
                    id-id is used for uniqueness which contains only single element.which is represented with #
                  

                   There are 4 types of color giving they are -
                   1.Hexadecimal color-specified with #RRGGBB --> Red,Green,Blue.
                   2.Hexadecimal colors with Transparency -- Specified with #RRGGBBAA  A-specifies the transparency of the color.
                   3.RGB colors-Which are red,green and blue which have the value range from 0 to 255.
                   4.RGBA colors-A means alpha which specifies the opacity or transparency.The range is between (0.0) to (1.0).
                  

                  The display block is used to display the content in block level by giving display:block; .
                  The display inline is used to display the content in inline level by giving display:inline; .
                         For inline elements the width and height are not applied and margin, padding also not applicable.
                  The display inline-block allows to set a width and height on the element.and also we can give paddings are margins.
              

                    Div-->div is a block level element which is used to display throughout the screen of the content width.
                    The div elements starts in always new line.
                    Span-->Span is a inline element which is used to display the exact content width.
                    Thr span element starts in same line.
                  

                Float:It is used to display the  overall container right or left.
                Text-align: Whereas the text-align is used to display the content in container right,left/center.
                

                Pseudo-classes:It is used to style the specific state of an element.
                syntax:selector:pseudo-class{
                  property:value;
                }
                Pseudo-elements:It is used to style the specific part of an element.
                syntax:selector:pseudo-element{
                  property:value;
                }
                

                Box model in css is nothing but a container which includes content,border,padding and margin.
                It is used to create design and layout of the web pages.
                

                Padding-Padding is nothing but giving the space between the content and the border inside element.
                margin-whereas margin is nothing but space outside the content.
                

                Position:relative-The position relative is positioned relative to its normal position.
                Position:absolute-whereas position absolute is positioned relative to its nearest ancestor.
                

                Position:fixed-The position fixed property is fixed to the viewport and doesn't move irrespective of scrolling
                Position:sticky-whereas position sticky property can fixed to a particular offset value given by the user. 
                

                Position-The position is used to display the content in left,right,top and bottom
                Display-whereas the display property is used to display the content in
                 display:block,display:inline,display:flex,display:grid,and display:inline.
                

                Transtition-It is used to change the property value from one state to other state.
                Transform-whereas transform property is used to change the appearance of the element.
                

                Transtition-duration-->The transtition duration is used to display how many sec the transition can be performed /
                how many sec the transtition takes to complete.
                Transtition-timing-function-->whereas the transition timining-function is used to display how speed the transtition can be performed.
                

                flex-flex is a one dimensional container.It is usedto display content in only one direction i.e horizontal direction.
                    The default direction of flex is row/horizontal.
                Grid-grid is a two dimensional container.It is used to display the content both in row and column wise at a time.
                     The default direction of grid is column/Vertical.
                

                Media queries is nothing but a tool which is used to design or style our webpage responsive.
                

               The responsivedesign is nothing but adjusting or adapting our styles to different screen sizez i.e for mobile,tablet,small laptop,large laptop..
                

                Z-index property specifies stack order of an elements.
                An element with greater stack order is always in front of an element with a lower stack order.
                

                align-items->It is used to to align the items in the container.
                align-content->Whereas align-content is used to align the overall container.
                

                  The media rule is used in media queries to apply different styles for different media types/devices.
                  There are four types of media in css they are;
                  1.screen->used for computer screens,tablet and smartphones.
                  2.print->used when printing a document
                  3.speech->used for screen readers that read the page content aloud.
                  4.all->applied to all devices.
                

                For flex->To display the div horizantally and vertically we have to give justify-content:center and align-items:center in 
                parent and specify height in parent.If height is not specified it won't work.
                ex:body{
                  height:100vh;
                  display:flex;
                  justify-content:center;
                  align-items:center;
                }
                .class{
                  background-color:white;
                  width:200px;
                  height:200px;
                }
                For grid->we have to give height and place-content:center in parent container.
                ex:body{
                  height: 100vh;
                  display: grid;
                  place-content: center;
                 }
                 .class{
                  background-color: aqua;
                  width: 300px;
                  height: 100px;
                 }
